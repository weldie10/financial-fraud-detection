# Notebooks

This directory contains Jupyter notebooks for exploratory data analysis, feature engineering, modeling, and explainability.

## Notebooks

- `eda-fraud-data.ipynb` - Exploratory Data Analysis for fraud data
- `eda-creditcard.ipynb` - Exploratory Data Analysis for credit card data
- `feature-engineering.ipynb` - Feature engineering pipeline
- `modeling.ipynb` - Model training and evaluation
- `shap-explainability.ipynb` - SHAP-based model explainability
