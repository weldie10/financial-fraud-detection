# Data Directory

This directory contains all datasets used in the project.

## Structure

```
data/
├── raw/              # Original, immutable source data
└── processed/        # Cleaned and transformed data
```

## Usage Guidelines

### Raw Data (`data/raw/`)

- **Purpose**: Store original, unmodified source datasets
- **Rule**: Never modify files in this directory
- **Contents**:
  - `Fraud_Data.csv` - E-commerce fraud dataset
  - `creditcard.csv` - Credit card fraud dataset
  - `IpAddress_to_Country.csv` - IP to country mapping

### Processed Data (`data/processed/`)

- **Purpose**: Store cleaned, feature-engineered, and transformed datasets
- **Rule**: Generated by preprocessing pipeline, can be regenerated
- **Contents**:
  - `processed_fraud_data.csv` - Preprocessed fraud data
  - `processed_creditcard_data.csv` - Preprocessed credit card data
  - `eda_outputs/` - EDA visualizations and reports
  - `transformers.joblib` - Saved preprocessing transformers

## Data Governance

- Raw data is **immutable** - never edit source files
- All transformations are reproducible via preprocessing pipeline
- Processed data can be regenerated from raw data
- Large files are excluded from git (see `.gitignore`)
